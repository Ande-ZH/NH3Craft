# 10 风控与安全

> 返回：[总目录](index.md)

## 为什么需要风控

NH3 的经济系统围绕 XP 展开。如果没有风控，少数玩家可以通过漏洞、小号、AFK 刷怪、洗钱等方式快速积累大量 XP，破坏经济平衡。风控的目标是：

- **让正常玩家长期发展**。
- **让漏洞路线无利可图**。
- **让寡头垄断付出代价**。

---

## 杀怪防刷机制

杀怪 XP 有多层防刷保护，确保收入来自真实游戏行为：

| 机制 | 参数 | 效果 |
| --- | --- | --- |
| 滚动窗口上限 | 1000 XP / 10 分钟 | 超过后当窗口内不再获得 XP |
| 区块衰减 | 同区块累计 >500 XP | 收益乘以 0.1（降为十分之一） |
| AFK 检测 | 2 分钟无移动 | 停止计入 XP |
| 刷怪笼惩罚 | 刷怪笼产出的怪 | XP × 0.5 |
| 玩家造傀儡 | 铁傀儡/雪傀儡 | 0 XP，完全不计 |
| Boss 日限额 | 500 XP / 天 | 超过后当日不再获得 Boss XP |
| Boss 周限额 | 2000 XP / 周 | 超过后当周不再获得 Boss XP |

**Boss XP（受日/周限额约束）**：
- 监守者：20 XP
- 凋灵：50 XP
- 末影龙：100 XP

**正常冒险不受影响**：切换区块可重置区块累计。

## 工业出口防刷机制

| 机制 | 说明 |
| --- | --- |
| 每日次数限制 | 每种物品有独立的每日出口次数上限 |
| Tier 门槛 | 高阶物品需要更高 Tier 才能出口 |
| 饱和度降价 | 全服同一种物品出口过多时，价格暴跌（最低 15%） |
| 经济政策调控 | 服务器过热时工业倍率自动下调 |

**建议**：不要全服一起卷同一种出口品，多元化出口品才是正道。

## 转账风控 (`/pay`)

| 机制 | 数值 |
| --- | --- |
| 转账税 | 3%（直接销毁，不进任何人钱包） |
| 最低转账额 | 10 XP |
| 最高转账额 | 按 Tier 限制（T1: 5万, T2: 10万, T3: 30万, T4: 100万, T5: 无限） |
| 冷却时间 | 同一对象 60 秒 |
| 每日上限 | 同一收款人 100,000 XP/天 |
| 同 IP 检测 | 直接禁止，并记录审计日志 |
| 72h 在线门槛 | 必须累计 72 小时有效在线才能使用 |

## 市场熔断

当检测到**异常大额交易**或**单品价格暴涨**时，系统自动暂停交易：

- **触发条件**：价格偏离基准价 **5 倍**。
- **处理方式**：交易冻结 **10 分钟**，全服广播警报。
- **目的**：防止误操作、洗钱、极端价格操纵。

## 领地与性能风控

| 行为 | 后果 |
| --- | --- |
| 大量圈地长期闲置 | 维护费持续扣除，欠费后领地保护冻结 |
| 同一区块堆大量机器 | TPS 下降，可能被要求整改 |
| 长期运行无人值守高频装置 | 性能压力，可能触发管理干预 |
| 故意制造掉落物/实体/红石卡顿 | 违规处理 |

---

## 经济健康自动调节

服务器会实时监控经济指标，自动调整政策：

| 指标 | 紧缩触发 | 宽松触发 | 效果 |
| --- | --- | --- | --- |
| 周增长率 | >15% | <3% | 杀怪/工业产出下调或上调 |
| 基尼系数 | >0.6 | — | 启动更高累进税 |
| CPI | 偏离超过 30% | — | 央行调整锚定价格 |
| 流入/流出比 | >1.5 | <0.7 | 下一小时产出微调 |

## 行为红线（会被处罚）

以下行为可能触发限制、冻结、回滚或管理员处理：

### 经济套利
- 小号给大号转移 XP 或物资。
- 用极端价格刷成交记录。
- 用市场伪装洗钱。
- 利用价格熔断漏洞套利。

### 刷钱与漏洞
- 利用 AFK、刷怪塔、插件漏洞绕过收益限制。
- 利用机器、范围挖掘或异常破坏刷矿物碎片。
- 利用 Slimefun 配方漏洞复制物品。
- 发现刷钱路径后不报告而持续利用。

### 市场操控
- 恶意囤积关键新手材料。
- 联合抬价或压价破坏市场。
- 用不合理价格诱导新玩家交易。
- 通过多个账号制造虚假成交量。

### 领地和性能
- 大量圈地长期闲置。
- 同一区块堆大量机器导致卡顿。
- 长期运行无人值守高频装置。
- 故意制造掉落物、实体或红石卡顿。

---

## 正常行为是被鼓励的

- 正常生产。
- 正常交易。
- 合作建厂。
- 合理定价和稳定供货。
- 报告漏洞和异常收益。

**风控不是限制正常玩家，而是保护正常玩家的长期利益。**
