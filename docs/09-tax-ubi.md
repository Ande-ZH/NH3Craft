# 09 税收、维护费与 UBI

> 返回：[总目录](index.md)

## 为什么有税

如果服务器只发 XP 不回收，后期会严重通胀。税费用于：

- 回收过量 XP。
- 支持 UBI 全民基础收入。
- 抑制少数玩家无限囤积。
- 稳定市场价格。
- 让大规模占地和商业经营承担成本。

## 税费总览

| 税种 | 触发时机 | 税率/金额 | 去向 |
| --- | --- | --- | --- |
| 转账税 | `/pay` 转账时 | 3% | 销毁 |
| 交易税 | QuickShop 商店交易 | 5% | 服务器 |
| 上架费 | AuctionHouse 上架 | 2% | 服务器 |
| 成交税 | AuctionHouse 成交 | 3% | 服务器 |
| 领地维护费 | 每周结算 | 见下文 | 销毁 |
| 财富税 | 每周日 23:00 结算 | 见下文 | 60% UBI + 40% 销毁 |

## 领地维护费

| 规则 | 数值 |
| --- | --- |
| 前 6 块 | 50 XP/块/周 |
| 第 7 块起 | 300 + (块数 − 6) × 100 XP/周 |
| T5 优惠 | 维护费 30% 返还（peak ≥ 10000） |

> 例如：4 块领地 = 200 XP/周，10 块领地 = 300 + 400 = 700 XP/周。领地不会因为一次欠费就立刻被删除，欠费后进入宽限期。

## 财富税

每周日 23:00 自动结算，按余额分级征收：

| 余额区间 | 税率 |
| --- | --- |
| ≤ 10,000 XP | 免税 |
| 10,001–50,000 XP | 1% |
| 50,001–200,000 XP | 3% |
| 200,001–1,000,000 XP | 6% |
| > 1,000,000 XP | 10% |

> 财富税是阶梯制：只对超出部分按对应税率征税，不是全额乘以最高税率。

### 财富税计算示例

余额 300,000 XP 的玩家：
- 0–10,000：免税
- 10,001–50,000：40,000 × 1% = 400 XP
- 50,001–200,000：150,000 × 3% = 4,500 XP
- 200,001–300,000：100,000 × 6% = 6,000 XP
- 合计：10,900 XP

## UBI 全民基础收入

税收收入分配：

| 去向 | 比例 |
| --- | --- |
| UBI 基金 | 60% |
| 销毁（通缩） | 40% |

### 领取条件

- 本周有效登录天数 ≥ 3 天。
- 本周有效在线时长 ≥ 2 小时。

### 分配规则

| 规则 | 数值 |
| --- | --- |
| 每人最低保底 | 50 XP |
| 分配比例 | 当周 UBI 基金的 60% 平分给合格玩家 |
| 结转 | 剩余 40% 滚入下周 UBI 基金 |

### UBI 计算示例

本周 UBI 基金 60,000 XP，50 位合格玩家：
- 实际分发：60,000 × 60% = 36,000 XP
- 每人分到：36,000 ÷ 50 = 720 XP
- 剩余滚入下周：60,000 × 40% = 24,000 XP
- 因为 720 > 50，保底不触发

## 玩家建议

- **普通玩家基本不用担心税**——10,000 XP 以下免税，财富税主要影响高余额玩家。
- 高玩应把 XP 投资到生产、交易和项目，而不是无限囤余额——余额越高，被税越重。
- 领地只保留真正在用的基地、工厂和项目用地——第 7 个起每个每周 100 XP。
- T5 玩家享受领地维护费 30% 返还，大规模圈地前先把 Tier 冲上去。
- 每周日 23:00 前可以适当消费或投资减少余额，降低财富税。
- UBI 是稳赚的被动收入，只要保持每周 3 天 × 2 小时以上在线即可领取。
