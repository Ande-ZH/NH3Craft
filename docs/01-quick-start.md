# 01 快速开始

> 返回：[总目录](index.md)

## 你刚进服应该做什么

第一次进入 NH3 Craft 时，不建议一上来就刷怪或大规模圈地。按以下步骤走最稳：

1. 正常生存，先确保食物、木头、石头和基础庇护所。
2. 输入 `/kit starter` 领取新手启动包，包含 **32 个铁锭**。
3. 挖矿时留意石头、深板岩、凝灰岩额外掉落的 **Slimefun 矿物碎片**（2% 概率）。
4. 使用 `/sfc <物品名>` 查看 Slimefun 物品合成材料。
5. 使用 `/sci` 查看当前区块 Slimefun 机器数量和性能影响。
6. 用 `/playtime` 关注你的有效在线时长——**72 小时**后才能解锁转账、拍卖和借款。
7. 先做基础机器，再考虑工业出口和市场交易。

## 新玩家最重要的三个概念

### 1. XP 是钱

XP 不只是经验等级，它是服务器的核心货币。你用 XP 进行：

- 玩家间转账（`/pay`）
- 市场购物与摊位租金
- 税费缴纳
- 工业出口结算

### 2. 历史峰值决定 Tier，不是余额

系统同时记录两个值：

| 概念 | 含义 | 主要影响 |
| --- | --- | --- |
| 当前 XP 余额 | 你现在还能花多少 | 转账、购物、还款、缴税 |
| 历史最高 XP（Peak） | 你曾达到过的最高值 | Tier 解锁、权限 |

**等级公式：LV = XP ÷ 200**（200 XP/级）。例如 1600 XP = Lv.8，5000 XP = Lv.25。

```text
你攒到 5000 XP → 历史峰值 = 5000 → 达到 Tier 3
你花掉 3000 XP → 当前余额剩 2000
但历史峰值仍为 5000 → Tier 3 不会掉
```

**核心规则：Tier 由历史最高 XP 决定，花 XP 不会降级。**

### 3. 挂机和刷塔不会无限赚钱

| 防刷机制 | 说明 |
| --- | --- |
| 杀怪 XP 滚动上限 | 每 10 分钟最多 1000 XP |
| 单区块饱和 | 同一区块累计超过 500 XP 后，杀怪收益降到 0.1 倍 |
| AFK 检测 | 2 分钟（120 秒）无移动 = 停止计算杀怪 XP |
| 刷怪笼怪物惩罚 | 刷怪笼产出的怪物 XP × 0.5 |
| 玩家制造铁傀儡/雪傀儡 | 0 XP |

正常游玩收益稳定，挂机套利会被压低收益或触发风控。

## 新手前 3 小时建议

- 做原版基础装备（石制→铁制）。
- 建一个小型安全基地。
- 挖铁、煤、铜，顺便收集矿物碎片。
- 学会查 Slimefun 配方（`/sfc`）。
- 不急着借款，不急着开大工厂。
- 注意：原版矿物已减少，煤/铁/铜只有 0.7 倍，钻石只有 0.5 倍。

## 新手前 72 小时目标

许多金融和市场玩法需要 **72 小时有效在线时长** 后才能使用：

| 解锁内容 | 72小时在线要求 |
| --- | --- |
| `/pay` 转账 | 需要 |
| 参与拍卖 | 需要 |

你的 72 小时目标：

- 形成稳定材料来源。
- 达到 Tier 1（1600 XP 峰值，约 Lv.8）。
- 了解 `/pay` 和市场条件。
- 建立不会亏本的基础生产线。

## 玩家建议

- 第一天别冲出去打怪，先把生存基础做扎实。
- 新手包的 32 铁锭不要乱花——这是你最早期的工业启动资金。
- 矿物碎片看起来少，但攒起来用机器处理是前期的核心补充收入。
- 用 `/playtime` 随时查你的有效在线时长，别到了 72 小时还不知道自己能干什么。
