# 07 战斗、杀怪 XP 与防刷限制

> 返回：[总目录](index.md)

## 杀怪 XP 是稳定收入

杀怪是二期重要 XP 来源之一（目标占比约 40%），但有明确上限和防刷机制。

## 杀怪 XP 详细表

| 怪物类型 | 触发概率 | XP | 备注 |
| --- | --- | --- | --- |
| 僵尸/骷髅/蜘蛛/苦力怕/溺尸/尸壳 | 30% | 1 | 普通怪，低收益 |
| 末影人/女巫/卫道士/掠夺者/幻翼 | 60% | 3-8（随机） | 中级怪 |
| 唤魔者/劫掠兽/猪灵 bruteforce | 100% | 15-25 | 精英怪 |
| 监守者 | 100% | 500 | **绕过限额** |
| 凋灵 | 100% | 1000 | **绕过限额** |
| 末影龙 | 100% | 2000 | **绕过限额** |
| 玩家造铁傀儡/雪傀儡 | 0% | 0 | 完全不给 XP |

## 防刷机制详解

| 机制 | 参数 | 效果 |
| --- | --- | --- |
| **滚动漫游上限** | 1000 XP / 10 分钟 | 超过后窗口内不再获得 XP |
| **区块衰减** | 同区块累计 >500 XP | 收益乘以 0.1（降为十分之一） |
| **AFK 检测** | 2 分钟无移动 | 停止计入 XP |
| **刷怪笼惩罚** | 刷怪笼产生的怪 | XP × 0.5 |
| **Boss 绕过** | 监守者/凋灵/末影龙 | 不计入窗口上限 |

### 举例

- 你在 10 分钟内杀了 1000 个僵尸（理论上 300 XP），但加上区块衰减和窗口上限，实际约拿到 300-1000 XP 左右。
- 打一只凋灵直接拿 1000 XP，且不计入窗口 — Boss 狩猎是突破性收入。

### 深夜衰减

凌晨 2:00-6:00 的在线时间按 **0.5×** 计入有效在线时长。杀怪 XP 本身不受影响，但这段时间的在线时长对 Tier 解锁打折。

### 切换区块恢复

当你移动到新区块时，该区块的累计 XP 重新从 0 开始计算。所以合理移动打怪比站桩刷怪收益更高。

## 活动加成

管理员可以开启双倍 XP 活动（`/nh3admin event doublexp`），活动期间所有杀怪 XP 翻倍。

## 正常打怪 vs 刷怪塔

| 方式 | 预期收益 | 风险 |
| --- | --- | --- |
| 正常冒险 | 稳定，Boss 爆发 | 无 |
| 刷怪笼 | XP × 0.5 | 收益减半 |
| 挂机打怪 | 2 分钟后停计 | AFK 检测 |
| 站桩刷怪 | 区块衰减到 0.1× | 极低收益 |

## 玩家建议

- 正常冒险和打 Boss 是最健康的收入方式。
- 不要把全部收入押在刷怪塔上 — 区块衰减和窗口上限会让收益锐减。
- Boss 是突破性收入：凋灵 1000 XP、末影龙 2000 XP 且不计限额。
- 换个区块打怪可以重置区块累计，比一直站桩强。
- ElectricSpawners、MobCapturer、SoulJars 等产出的怪受刷怪笼 0.5× 惩罚。
