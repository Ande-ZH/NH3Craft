# 13 宏观经济：CPI、Gini 与政策变化

> 返回：[总目录](index.md)

## 为什么玩家需要知道宏观经济

你不需要会算 CPI 或 Gini，但这些指标会影响：

- 杀怪收益。
- 工业出口倍率。
- 税率。
- UBI。
- 市场价格和熔断。

## Gini 基尼系数

Gini 衡量财富分配是否失衡。

| Gini | 含义 |
| --- | --- |
| 0 | 所有人一样富 |
| 0.25–0.40 | 偏平均 |
| 0.35–0.55 | 较健康 |
| 0.60+ | 贫富差距明显 |
| 0.70+ | 寡头化风险高 |

如果少数玩家囤积大量 XP，系统可能提高税收、加强回收或增加 UBI。

> 服务器 Gini 目标区间：**0.35–0.55**。偏高时财富税与 UBI 自动加强。

## CPI 物价指数

CPI 衡量核心物资价格变化。

篮子可能包含：

- 铁、铜、煤、红石。
- 钢、铝、红石合金。
- 能源材料。
- 高阶工业材料。

CPI 上升表示物价上涨，同样 XP 能买到的东西变少。

## 经济政策

服务器可能自动切换政策：

| 触发条件 | 状态 | 政策效果 |
| --- | --- | --- |
| XP 总量周增长 > 15% | 通胀风险 | **紧缩**：杀怪 ×0.85，工业 ×0.9，税率 ×1.2 |
| XP 总量周增长 < 3% | 经济过冷 | **宽松**：杀怪 ×1.1，工业 ×1.1，税率 ×0.9 |
| 流入/流出比 > 1.5 | 短时通胀 | 下小时产出下调 10% |
| 流入/流出比 < 0.7 | 短时通缩 | 下小时产出上调 10% |
| 3% ≤ 增长 ≤ 15% | 基本平衡 | 维持参数 |

## 玩家会感受到什么

- 同样怪物有时 XP 变少。
- 同样出口品价格波动。
- 税率或维护成本变化。
- UBI 发放金额变化。
- 市场异常价格被熔断。

## 玩家建议

- 不要把收益变化立刻当成 bug，可能是经济政策调整。
- 商人应关注公告、市场价格和核心材料趋势。
- 工业玩家应准备多条出口路线，避免单品饱和。
