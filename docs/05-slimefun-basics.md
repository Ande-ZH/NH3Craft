# 05 Slimefun 入门与常用工具

> 返回：[总目录](index.md)

## Slimefun 是核心

Slimefun 是服务器的科技核心。原版矿物减少后，你几乎必须走 Slimefun 路线才能高效产出 XP。

## 硬核设定

| 项目 | 设定 | 说明 |
| --- | --- | --- |
| 研究成本 | 1.5× | HardcoreSlimefun 开启研究倍率 |
| 机器故障率 | 3% | HCSF 机器故障，需要 SF 扳手修复 |
| 工具耐久 | 0.75× | 工具耐用度降低 |
| 全局机器耗电 | 2.5× | 所有机器耗电量是原版 2.5 倍 |
| 配方难度 | 1.5× | HCSF 提升合成难度 |

## 电力系统重点

服务器对电力做了分层平衡：

### 发电机输出倍率

| 时期 | 代表机器 | 输出倍率 | 说明 |
| --- | --- | --- | --- |
| 早期 | 煤炭/生物质/岩浆/镁发电机 | 1.0× | 鼓励新手使用 |
| 中期 | 太阳能2-3/半流体/地热/蒸汽 | 0.7× | 平衡期 |
| 后期 | 太阳能4/核反应/燃烧/燃气涡轮 | 0.5× | 压制膨胀 |
| 终局 | HV太阳能/LX核能/虚空采集 | 0.4× | 严厉压制 |
| HV核反应堆 | HV_NUCLEAR_REACTOR | 0.35× | 性价比最高的额外压 |

### 机器耗电分类

| 机器类型 | 耗电系数 | 实际耗电 | 说明 |
| --- | --- | --- | --- |
| 入门机器 | 0.8× | 2.5×0.8=2.0× | 电炉、矿石粉碎机、自动附魔台 |
| 普通机器 | 1.0× | 2.5×1.0=2.5× | 大多数机器 |
| 高耗能后期机器 | 1.5× | 2.5×1.5=3.75× | 反物质合成器、无限熔炉 |

**影响**：新手很难被电力卡死，但后期想无限堆核反应堆会被 0.35-0.4× 压制。

## 研究与权限锁

部分高阶机器需要 Tier 权限才能研究和合成：

| 机器 | 最低 Tier | 研究成本 |
| --- | --- | --- |
| 核反应堆 | T3（5000 XP） | 100 XP × 1.5 = 150 XP |
| GEO 扫描仪 | T3 | 高研究成本 |
| 反物质反应堆 | T4（8000 XP） | 500 XP × 1.5 = 750 XP |
| Infinity 相关 | T4 | 8000 XP 研究成本 |
| HV 核能（LiteXpansion） | T4+ | 0.4× 发电 + 0.75× 核棒效率 + 更严苛 |

## HCSF 故障系统

机器有 3% 基础概率故障，故障后需要 **SF 扳手** 修复。这是硬核设定的一部分——设计合理的生产线应该有冗余。

## LiteXpansion 核能额外压制

LiteXpansion 核能受到三重压制：

| 压制层 | 效果 |
| --- | --- |
| EcoPower 发电倍率 | ×0.4 |
| 核棒效率 | ×0.75 |
| 反应堆网格限制 | 最大 5×5（原版 6×6） |
| 核棒寿命 | 15000 tick（原版 30000） |
| 爆炸 | 启用，半径 12 格 |

## 玩家建议

- 新手优先做电力熔炉和矿石粉碎机（0.8× 耗电折扣）。
- 煤炭发电机是前期最稳定的电力来源（1.0× 发电）。
- 不要急着做核反应堆——维护成本和耗电都很高。
- 用 `/sfc` 算好成本再开工，避免浪费材料。
- 故障率 3% 不高，但关键产线建议做冗余。
