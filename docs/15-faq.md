# 15 FAQ 与避坑指南

> 返回：[总目录](index.md)

## 为什么 `/pay` 用不了

可能原因：

- Tier 不够。
- 有效在线不足 72 小时。
- 金额太低或超上限。
- 冷却未结束。
- 被风控判定为异常转账。

## 为什么挖矿没掉碎片

可能原因：

- 概率未触发。
- 你处于创造模式。
- 使用了机器或异常破坏方式。
- 触发冷却中。
- 该方块不在补偿列表。

## 为什么杀怪没 XP

可能原因：

- AFK。
- 刷怪笼怪物收益衰减。
- 达到个人或区块上限。
- 经济政策降低收益。
- 怪物类型无奖励。

## 为什么出口价格变低

可能原因：

- 该物品全服出口过多。
- 当前经济处于紧缩。
- 你的 Tier 未解锁更高阶路线。
- 出口品价格被手动调参。

## 为什么不能去 Nether 或 End

维度受到 Tier 锁控制。先提升历史 XP 和 Tier。

## 高玩是不是会越来越富

高玩有生产优势，但系统通过财富税、摊位租金、出口饱和、市场熔断和 UBI 抑制差距扩大。

## XP 应该存着还是花掉

达到关键 Tier 前可以适当存。达到关键 Tier 后，投资生产线、市场和材料通常比无限囤余额更有价值。
