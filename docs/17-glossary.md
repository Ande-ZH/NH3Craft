# 18 术语表

> 返回：[总目录](index.md)

| 术语 | 含义 |
| --- | --- |
| XP | 核心货币，不只是经验等级 |
| 当前余额 | 你现在能花的 XP |
| 历史峰值 / peak | 你曾经达到过的最高 XP |
| Tier | 根据历史峰值等因素决定的成长阶层 |
| 有效在线 | 排除挂机后的在线时间 |
| AFK | 长时间无有效活动 |
| 工业出口 | 上交指定 Slimefun 物品换 XP |
| 饱和度 | 某物品全服出口过多导致价格下降的指标 |
| CPI | 物价指数，衡量核心材料价格变化 |
| Gini | 基尼系数，衡量财富分配差距 |
| UBI | 全民基础收入，税收基金返还活跃玩家 |
| 领地维护费 | 持有领地的周期性成本 |
| 财富税 | 高余额玩家周期性税费 |
| 熔断 | 市场价格异常时暂停交易或记录异常 |
| NH3SFBridge | 服务器内部用于识别/扣除/发放 Slimefun 物品的桥接插件 |
| `/sfc` | Slimefun 合成查询命令 |
| `/sci` | Slimefun 区块机器信息查询命令 |
