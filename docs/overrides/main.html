XP 核心经济

二期把 XP 作为核心货币，余额用于支付、交易、税费和还款，历史最高 XP 则决定长期 Tier 与成长进度。

⚙️

Slimefun 工业循环

挖矿获得矿物碎片，投入粘液科技机器生产工业品，再通过出口和玩家市场获得 XP，形成可持续生产链。

📈

市场与宏观调控

出口饱和、价格走廊、市场熔断、CPI 与 Gini 指标共同维持经济稳定，避免无限刷钱和寡头垄断。

🧭

Tier 阶层解锁

权限、维度、市场、转账和高级机器会随 Tier 逐步开放。先达到 Tier 1 与 72 小时有效在线，是新玩家第一目标。

🏦

借款与信用系统

借款不再是免费 XP，而是基于有效在线、历史等级、信用分和还款记录的金融工具，适合用于明确可回本的扩建。

🛡️

风控与长期环境

转账税、领地维护费、财富税、UBI、反小号套利与异常交易记录，让正常玩家长期发展，让漏洞路线无利可图。

二期成长路线图

从新手生存到工业寡头，你的进度不只取决于刷资源，
还取决于 XP 峰值、生产线、信用记录和市场判断。

Phase I : 新手生存与基础经济
01
🏁

进入服务器

/sfc/sciStarter

首次进入领取 32 个铁锭启动包，学习基础命令，建立食物、矿物和安全住所，不急着借款或大规模圈地。

02
⛏️

挖矿与矿物补偿

OreChunksMining

原版矿物产出会被压低，但挖石头、深板岩、下界岩和矿物时可能获得 Slimefun 矿物碎片，工业玩家长期收益更高。

03
⚔️

战斗 XP

Kill XPAnti-AFK

正常冒险和战斗能获得 XP，但 AFK、刷怪笼、同区块高频产出和个人窗口上限会限制刷怪塔收益。

Phase II : 公民、市场与工业生产
04
🧱

Tier 1 与有效在线

Tier72h/playtime

通过历史最高 XP 与有效在线解锁基础经济功能。达到 Tier 1 和 72 小时有效在线后，转账、市场与借款资格才逐步开放。

05
🏭

工业出口

/nh3exportSlimefun

将指定 Slimefun 工业品出售给系统换取 XP。物品价格会受 Tier、每日次数、服务器饱和度和经济政策影响。

06
🛒

玩家市场

/payShopAuction

玩家可以通过转账、摊位、拍卖和自由交易流通资源。极端价格、洗钱、小号转移和借款逃债会触发风控。

07
⚡

分层电力

EcoPower/sci

前期发电基本正常，中后期和终局能源会被压低效率。合理规划机器区块、线路和生产节奏，比无脑堆机器更重要。

Phase III : 高阶经济与长期经营
08
🏦

信用借款

LoanCredit

信用分、历史 XP、有效在线和还款记录决定额度与利率。借款期间禁止资产转移，逾期会进入违约或破产追缴。

09
📊

宏观经济

CPIGiniUBI

服务器会观察 XP 流入流出、CPI 和 Gini，在经济过热或冷清时调整收益、税率和 UBI，让长期生态更稳定。

10
👑

工业寡头

Endgame Economy

高阶玩家拥有更强生产能力，也承担更高财富税、摊位租金和市场责任。稳定供货、合理定价和产业协作才是长期优势。
