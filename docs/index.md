## 新玩家最短路线

1. 阅读 [快速开始](01-quick-start.md)。
2. 理解 [XP 经济](02-xp-economy.md) 和 [Tier 解锁](03-tier-unlock.md)。
3. 学会 [挖矿与矿物补偿](04-mining-and-orechunks.md)、[Slimefun 入门](05-slimefun-basics.md)。
4. 通过 [工业出口](06-industry-export.md)、[战斗 XP](07-combat-xp.md)、[市场交易](08-market-trading.md) 赚钱。

## 核心循环

```text
挖矿 / 战斗 / Slimefun 生产
→ 获得材料或 XP
→ 提升历史峰值 XP 与 Tier
→ 解锁市场、维度和更高阶机器
→ 通过出口、交易和生产线形成长期收入
→ 税收、维护费、饱和度和市场规则回收 XP
```
