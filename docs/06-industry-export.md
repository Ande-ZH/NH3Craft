# 06 工业生产与出口赚钱

> 返回：[总目录](index.md)

## 什么是工业出口

工业出口是把指定 Slimefun 工业品交给系统换取 XP 的核心玩法。

```text
生产指定工业品
→ /nh3export sell <物品ID>
→ 系统检查 Tier、数量、限额、饱和度
→ 扣除物品，发放 XP
```

## 饱和度机制

全服同一种物品出口越多，价格越低：

| 饱和度（全服出口次数/阈值） | 实际价格 |
| --- | --- |
| ≤ 1.0 | 100%（原价） |
| 1.0 – 2.0 | 70% |
| 2.0 – 3.0 | 40% |
| > 3.0 | 15% |

**含义**：如果全服都在卖铜锭，你的收益只有 15%。多元化出口品才是正道。

## 出口检查流程

1. **Tier 检查** — 你的历史峰值是否达到该物品要求的等级？
2. **背包检查** — 你是否有足够的 SF 物品？（通过 nh3sfbridge 插件精确识别）
3. **每日限额** — 你今天这个物品出口了几次？
4. **饱和度** — 全服今天这个物品出口了多少？
5. **经济政策** — 服务器当前是否处于紧缩期？

## 产出公式

```
最终价格 = 基础价 × 饱和度系数 × 经济政策系数
```

经济政策系数由服务器自动调节：
- 紧缩期：`industry_rate = 0.9`（打 9 折）
- 正常期：`industry_rate = 1.0`
- 宽松期：`industry_rate = 1.1`（加 10%）

## 玩家建议

- **不要全服一起卷同一种出口品** — 饱和度会让价格暴跌到 15%。
- 做生产线前用 `/sfc` 算好材料成本和电力成本。
- 低价出口品日限额高，适合每天稳定收入。
- 高价出口品日限额低（T5 每日仅 1 次），但单次利润高。
- 如果出口价突然变低，看看是不是饱和度触发了紧缩。
- `/nh3export sell <物品ID>` 用的是 Slimefun 物品 ID，用 `/sfc` 查到后直接输入。
