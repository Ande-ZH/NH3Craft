# 杀怪、挖矿与矿物补偿

> 返回：[模块目录](index.md)

## 5. 杀怪 XP

### 5.1 基础规则

杀怪可以获得 XP，但不是无限刷：

- 普通怪物通常是小额、概率性产出。
- 精英怪/高威胁怪物奖励更高。
- Boss 奖励最高，并且通常绕过普通额度限制。

示例设计口径：

| 类型 | 示例 | 奖励倾向 |
| --- | --- | --- |
| 普通怪 | 僵尸、骷髅、蜘蛛、苦力怕 | 小额概率奖励 |
| 中高威胁怪 | 末影人、女巫、卫道士 | 更高概率与更高 XP |
| 高价值怪/Boss | 袭击怪、凋灵、末影龙、监守者 | 高额奖励 |

### 5.2 防刷限制

为了防止刷怪塔破坏经济，杀怪 XP 会受到多种限制：

- **AFK 检测**：长时间无活动时不应继续有效产出。
- **刷怪笼衰减**：来自刷怪笼的怪物收益可能降低。
- **个人时间窗口上限**：例如 10 分钟内最多获得一定 XP。
- **区块产出衰减**：同一区块短时间产出过高会降低收益。
- **Boss 例外**：Boss 通常不受普通额度限制。

玩家建议：正常冒险和战斗收益稳定；长期挂机刷怪、集中刷塔会被系统压低收益。

## 6. 挖矿与矿物补偿

### 6.1 为什么矿变少了

二期会降低原版矿物的自然产出，目标是让工业链更重要，而不是让玩家只靠原版挖矿毕业。

大致方向：

| 矿物类型 | 调整倾向 |
| --- | --- |
| 煤、铁、铜等基础矿 | 约 0.7 倍 |
| 金、红石、青金石等中级矿 | 约 0.6 倍 |
| 钻石、绿宝石等高级矿 | 约 0.5 倍 |
| 下界/末地稀有资源 | 约 0.45 倍 |

### 6.2 Slimefun 矿物补偿

原版矿减少后，系统会通过 SlimefunOreChunks 或类似机制补偿工业玩家：

```text
原版矿物减少
→ 挖石头/深板岩/下界岩/原矿时有概率获得 SF 矿物碎片
→ 使用粘液科技机器提炼
→ 工业玩家获得更高长期收益
```

玩家侧表现通常是：

- 挖石头、深板岩、凝灰岩、下界岩等填充方块时，可能额外掉落矿物碎片。
- 挖原版矿物时，可能额外获得一份 SF 碎片奖励。
- 创造模式、机器挖掘或异常破坏通常不会触发奖励。
- 为保护性能，速挖时可能有短冷却，不是每个方块都会判定。

### 6.3 玩家建议

- 不要因为原版矿少就放弃挖矿；二期鼓励把挖到的碎片送进粘液科技生产线。
- 前期重点积累铁、铜、煤和基础机器。
- 中期开始把粉末/碎片转化成可出口或可交易的工业品。
