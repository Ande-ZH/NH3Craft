## 15. 常用命令速查

| 命令 | 用途 |
| --- | --- |
| `/sfc <物品名>` | 查看粘液科技物品合成材料 |
| `/sci` | 查看当前区块机器数量/性能影响 |
| `/playtime` | 查看有效在线时长 |
| `/pay <玩家> <金额>` | 给玩家转账，需满足解锁条件 |
| `/nh3export sell <物品ID>` | 出口指定工业品换 XP |
| `/nh3macro status` | 查看宏观指标，若服务器开放 |
| `/nh3stat report` | 查看经济报告，若服务器开放 |
| `/nh3top wealth` | 查看财富榜，若服务器开放 |

实际可用命令以服务器最终开放权限为准。

## 16. 推荐发展路线

### 16.1 新手期

目标：活下来，理解基础系统。

- 领取新手启动包。
- 用 `/sfc` 学基础机器。
- 挖矿收集原版矿和 SF 碎片。
- 不急着大规模圈地。

### 16.2 公民期

目标：解锁基础经济。

- 累计 72h 有效在线。
- 达到 Tier 1 / Lv.8 附近。
- 开始使用 `/pay`、市场和小规模交易。
- 保持良好信用记录。

### 16.3 商人期

目标：形成稳定收入。

- 建立可持续生产线。
- 选择不过度饱和的出口品。
- 开始经营摊位或参与拍卖。
- 控制领地数量，避免维护费过高。

### 16.4 工业期

目标：高阶 Slimefun 与跨维度扩张。

- 规划电力，不要盲目堆终局发电。
- 关注 CPI、出口饱和度和市场需求。

### 16.5 高阶/寡头期

目标：长期经营和承担经济责任。

- 高余额会面对更高财富税。
- 大量摊位和领地会有更高维护成本。
- 市场操纵、异常价格更容易被风控。
- 可以通过稳定供货、合理定价和产业协作获得长期优势。

## 17. 避坑 FAQ

### Q1：为什么我杀怪没有 XP？

可能原因：

- 你处于 AFK 或近期没有有效活动。
- 怪物来自刷怪笼，收益被衰减。
- 你达到了个人短时间收益上限。
- 同一区块产出过高，被区块衰减。
- 当前经济政策下杀怪倍率下降。

### Q2：为什么我挖矿感觉原版矿少了？

这是二期设计的一部分。原版矿会减少，但会通过 Slimefun 矿物碎片、粉末和工业生产线补偿长期收益。

### Q3：为什么 `/pay` 用不了？

可能原因：

- 未达到 Tier/等级要求。
- 有效在线不足 72h。
- 金额太低、超出上限或还在冷却。
- 同 IP/异常转账被风控拒绝。

### Q4：为什么我的出口价格变低了？

可能原因：

- 该物品全服出口过多，饱和度上升。
- 当前经济过热，工业倍率被下调。
- 你没有达到更高 Tier 的高价出口路线。

### Q5：为什么市场交易被冻结？

某个物品价格可能严重偏离参考价，触发熔断。等待熔断结束或调整为合理价格。

### Q6：为什么我不能去 Nether 或 End？

维度进入受 Tier 锁控制。Nether/跨维度通常需要中级 Tier，End 通常需要更高 Tier。

### Q7：为什么有效在线不涨？

可能原因：

- 2 分钟无活动被判定 AFK。
- 凌晨 2–6 点按较低倍率计入。
- 当日已达到 12 小时有效在线上限。

### Q9：高玩是不是会越来越富？

高玩有生产优势，但系统用财富税、摊位租金、出口饱和、市场熔断和 UBI 抑制差距扩大。Gini 过高时，政策会偏向回收和再分配。

### Q10：我该把 XP 存着还是花掉？

历史最高 XP 会记录你的阶层进度，因此达到关键 Tier 后，适当投资生产线、市场和扩建通常比无限囤余额更有价值。
