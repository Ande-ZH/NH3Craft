# 市场、摊位与宏观指标

> 返回：[模块目录](index.md)

## 12. 市场、摊位与拍卖

### 12.1 自由市场

服务器鼓励玩家自由交易，但会用宏观规则防止极端操控。

常见市场插件/形式包括：

- 玩家商店/摊位。
- 拍卖行。
- 中央银行价格走廊。
- 市场熔断。
- 经济公告牌/周报。

### 12.2 中央银行价格走廊

部分核心物资会有参考价区间：

```text
系统收购价 = 价格下限
系统出售价 = 价格上限
玩家自由市场 = 在上下限之间波动
```

它的作用是：

- 防止基础材料价格归零。
- 防止关键材料被炒到新玩家完全买不起。
- 给市场一个大致锚点。

### 12.3 市场熔断

如果某个物品价格严重偏离参考价，例如高于 5 倍或低于 1/5，系统可能触发熔断：

- 暂停该物品交易一段时间。
- 取消异常成交。
- 广播或记录市场异常。

玩家建议：不要用极端价格刷成交记录、洗钱或转移资产，这类行为很容易触发风控。

### 12.4 摊位租金

摊位/商店不是完全免费的长期仓库。

常见规则：

- Tier 越高，可持有摊位越多。
- 摊位越多，每周租金越高。
- 欠租可能进入宽限期。
- 长期欠租的商店可能被移除或冻结。

示例摊位数量方向：

| 阶段 | 摊位能力倾向 |
| --- | --- |
| 新手 | 通常 0 或很少 |
| Tier 1 | 少量摊位 |
| Tier 2 | 可经营小型商业 |
| Tier 3 | 可经营较大规模商店 |
| Tier 4+ | 大型商业玩家 |

## 14. 宏观指标：Gini、CPI 和经济政策

### 14.1 Gini 基尼系数

基尼系数衡量全服财富分配是否失衡：

| Gini | 含义 |
| --- | --- |
| 0 | 所有人一样富 |
| 0.25–0.40 | 偏平均 |
| 0.35–0.55 | 较健康，有差距但未失控 |
| 0.60+ | 贫富分化明显 |
| 0.70+ | 极危险，寡头化 |

玩家不需要计算它，只需要理解：如果少数人囤积大量 XP，系统可能提高税收、增加回收或调整 UBI。

### 14.2 CPI 物价指数

CPI 衡量核心物资价格是否上涨或下跌。服务器会选择一篮子常见材料，例如：

- 铁、铜、煤、红石。
- 钢、铝、红石合金。
- 能源水晶、反应堆核心。
- 终局工业材料。

如果 CPI 上升，说明同样的 XP 买到的东西变少了，也就是通胀。

### 14.3 自动政策

系统可能根据 XP 流入/流出、Gini、CPI 自动切换政策：

| 经济状态 | 可能政策 |
| --- | --- |
| XP 产出过高、物价上涨 | 紧缩：降低杀怪/工业收益，提高税率 |
| XP 回收过强、市场冷清 | 宽松：提高收益，降低部分税率 |
| 基本平衡 | 维持当前参数 |

玩家会感受到：同样行为的 XP 收益或税费可能随全服经济状态轻微变化。
