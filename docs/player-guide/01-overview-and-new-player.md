# 总览与新玩家开局

> 返回：[模块目录](index.md)

## 1. 二期的核心玩法一句话说明

NH3 Craft 二期把服务器经济从”单纯刷资源/刷经验”改成了一个更完整的循环：

```text
挖矿与粘液科技生产 → 出口/交易获得 XP → XP 决定等级 → 解锁更多玩法 → 税收/维护费/市场调控回收 XP
```

你需要理解三件事：

1. **XP 是核心货币**：大部分经济行为都围绕 XP 余额、XP 流水和历史最高 XP 展开。
2. **历史最高 XP 决定阶层/Tier**：即使你花掉 XP，已经达到过的高度仍会作为长期等级依据。
3. **经济不是无限放水**：杀怪、工业出口、市场都会有限额、税费或风控，防止刷钱和寡头垄断。

## 2. 新玩家开局

### 2.1 首次进入

首次进入服务器时，你会收到新手引导信息，通常会提示：

- `/sfc`：查看粘液科技合成需求。
- `/sci`：查看当前区块 Slimefun 机器数量与性能影响。
- `/playtime`：查看有效在线时长。

二期方案中还设计了新手启动包：

- **32 个铁锭**作为开局材料支持。
- 部分版本还会配合一本入门说明书或新手 GUI。

### 2.2 新手阶段建议

新手前期不要只盯着刷怪。推荐路线：

1. 先正常生存，建立基础矿物和食物来源。
2. 使用 `/sfc` 查粘液科技基础机器的材料。
3. 留意挖矿时额外获得的 Slimefun 矿物碎片/粉末。
4. 逐步进入基础工业生产，再考虑出口和市场交易。
5. 通过 `/playtime` 累计有效在线，为后续 `/pay` 和拍卖解锁做准备。

## 3. XP 货币与余额

### 3.1 XP 的作用

在二期里，XP 不只是经验等级，而是核心经济单位。它会影响：

- 你的可用余额。
- 你的历史最高财富记录。
- Tier/等级锁解锁。
- 借款额度和信用评估。
- 市场交易、税收和维护费。

### 3.2 余额和历史最高的区别

| 概念 | 含义 | 影响 |
| --- | --- | --- |
| 当前 XP 余额 | 你现在还能花的钱 | 支付、还款、税费、购买 |
| 历史最高 XP / peak | 你曾经达到过的最高财富 | Tier、等级荣誉、权限解锁、信用加成 |

也就是说，花钱不会立刻让你失去曾经达到过的阶层荣誉，但如果余额不足，你仍然不能支付、还款或缴税。

---

## 4. 如何获得 XP

二期主要有三类经济来源：

```text
杀怪产出 + 工业出口 + 玩家交易
```

设计目标大致是：

- 杀怪：约 40%。
- 工业出口：约 40%。
- 玩家交易：约 20%。

实际倍率会根据服务器经济健康度自动调整。
