## 7. 工业出口

### 7.1 什么是工业出口

工业出口是把指定 Slimefun 工业品交给系统，换取 XP 的玩法。它是最重要的稳定收入来源之一。

基本流程：

```text
生产指定工业品
→ 执行 /nh3export sell <物品ID>
→ 系统检查你的 Tier、物品数量、每日限额
→ 扣除物品
→ 发放 XP
```

### 7.2 出口会受到哪些限制

工业出口不是所有物品都能无限卖：

- 每种出口物品有指定的基础价格。
- 每种物品可能要求一定 Tier。
- 每日可出口次数有限。
- 单次需要指定数量。
- 全服同类物品出口过多时，会因“饱和度”降价。
- 服务器经济过热时，工业产出倍率可能下调。

示例饱和度逻辑：

| 饱和度 | 价格倾向 |
| --- | --- |
| ≤ 1.0 | 原价 |
| 1.0–2.0 | 约 70% |
| 2.0–3.0 | 约 40% |
| > 3.0 | 约 15% |

### 7.3 出口路线建议

- 前期：基础锭、粉末、铜/铁相关材料。
- 中期：钢、合金、红石合金、硫酸盐等工业材料。
- 后期：反应堆核心、Addons 高级材料、Infinity 类材料。
- 不要全服都挤同一种出口品，饱和会导致收益明显下降。

## 8. 玩家转账 `/pay`

### 8.1 解锁条件

设计中，`/pay` 不再是新号立刻可用的功能。常见要求包括：

- 达到指定 Tier/等级，例如 Lv.8 或 Tier 1。
- 有效在线达到 **72 小时**。

### 8.2 转账限制

`/pay` 通常会有以下限制：

- 最小金额，例如 10 XP。
- 单笔上限，随 Tier 提高。
- 1 分钟左右冷却。
- 每日同对象转账上限。
- 同 IP 或疑似小号转账可能被拒绝。
- 每笔转账收取交易税，例如 3%。

示例：你转账 1000 XP，若交易税 3%，对方实际收到约 970 XP，30 XP 进入回收/基金/销毁逻辑。

### 8.3 为什么要限制转账

这些限制不是为了麻烦玩家，而是为了防止：

- 小号养大号。
- 工作室批量洗钱。
- 超大额交易绕过市场审计。

正常玩家之间的小额合作、AA、购买材料通常不会受到影响。
