    12 命令教学：从进服到赚钱¶

    返回：总目录

    本页是 NH3 Craft 二期的命令教学，不再讲插件原理，只教你：什么时候、用什么命令、怎么用。

    建议把本页当工具书收藏，遇到具体场景回来查。

    第一节：刚进服的 5 分钟¶

    1. 领取新手启动包¶

    /kit starter

    首次进服立刻执行。你会获得 32 个铁锭——这是最早的工业启动资金，不要乱花，用来制作第一批 Slimefun 基础机器（电力熔炉、矿石粉碎机）。

    2. 查看你现在的进度¶

    /playtime

    显示你的有效在线时长（不是挂机时长）。记住这个关键数字：

    72 小时有效在线后，才能解锁 /pay 转账、拍卖和借款。

    3. 学会两个最常用的查询命令¶

    命令 什么时候用
    /sfc <物品名> 建产线前查合成材料。永远先查再动手，避免浪费材料。
    /sci 建工厂前、堆了很多机器后查当前区块的机器数量和性能影响。

    第二节：赚钱的命令（XP 从哪来）¶

    1. 杀怪（自动到账）¶

    杀怪不需要输入命令，XP 会自动发放。但你必须理解限制，否则会觉得"突然不给 XP 了"：

    • 每 10 分钟最多 1000 XP（滚动窗口）。
    • 同一区块累计超 500 XP → 收益降到 0.1 倍。换个区块就重置。
    • AFK（2 分钟不动） → 不计 XP。
    • Boss 收益高且不计上限：监守者 500、凋灵 1000、末影龙 2000。

    详见 战斗 XP 与防刷限制。

    2. 工业出口（核心收入）¶

    把 Slimefun 工业品卖给系统换 XP：

    /nh3export sell <物品ID>

    怎么用：

    1. 用 /sfc <物品名> 查到该物品的 Slimefun ID（大写、下划线，如 COPPER_INGOT）。
    2. 确认你的背包里有足够数量（出口会按“每次数量”扣除）。
    3. 执行 /nh3export sell COPPER_INGOT。

    示例：

    /nh3export sell COPPER_INGOT # 卖铜锭，每次扣 64 个，换约 15 XP /nh3export sell STEEL_INGOT # 卖钢锭，每次扣 16 个，换约 80 XP（需 Tier 1） /nh3export sell SULFATE # 卖硫酸盐，每次扣 32 个，换约 200 XP（需 Tier 2）

    常见失败原因：

    • 等级不足 → 该物品需要更高 Tier（详见 工业出口）。
    • 物品不足 → 背包里的 Slimefun 物品数量不够。
    • 今日次数用完 → 每个物品有每日出口次数上限。
    • 价格变低 → 全服都在卖这个，触发饱和度降价（可能只剩 15% 原价）。

    3. 卖给玩家 / 转账¶

    当你达到 Tier 1 + 72h 有效在线后，可以转账：

    /pay <玩家> <金额>

    • 扣 3% 转账税（直接销毁，不进任何人钱包）。
    • 同一对象 60 秒只能转一次。
    • 同 IP 不能转账（防小号套利）。

    第三节：升级你的 Tier（成长阶级）¶

    Tier 由历史最高 XP 决定，花 XP 不会降级。你不需要手动输入命令升级——达标自动晋升，并发放一次性奖励（Tier × 500 XP）。

    Tier 历史 XP 解锁
    T0 0 生存、杀怪 XP、出口 T1 品
    T1 1,600（Lv.8） /pay、拍卖买入、出口 T2 品、3 个摊位
    T2 3,000（Lv.15） 跨维度传送、拍卖卖出、出口 T3 品、10 个摊位
    T3 5,000（Lv.25） 末地、核反应堆、出口 T4 品、30 个摊位
    T4 8,000（Lv.40） 反物质/Infinity、无限额交易、出口 T5 品、60 个摊位
    T5 10,000（Lv.50） 税收返还 30%、150 领地、100 摊位

    完整权限矩阵见 Tier 与解锁。

    第四节：领地与保护¶

    /claim # 圈地保护（GriefPrevention） /trust <玩家> # 允许某玩家进入/使用你的领地

    领地越多维护费越高：

    领地数 维护费
    前 6 块 50 XP/块/周
    第 7 块起 300 + (块数-6) × 100 XP/周

    超过 Tier 配额无法继续圈地（T0=6, T1=10, … T5=150）。详见 税收与 UBI。

    第五节：查询经济信息¶

    命令 用途
    /nh3top wealth 财富排行榜
    /nh3macro status 宏观指标（CPI、Gini，需服务器开放）
    /ecocheck 查看当前物品的发电/耗电倍率（工业玩家必备）

    第六节：命令排错速查¶

    现象 先检查
    /pay 没反应 Tier 是否到 T1？有效在线是否到 72h？金额是否 <10 或超上限？是否同 IP？冷却中？
    /nh3export 失败 物品 ID 是否正确（大写+下划线）？背包数量够？Tier 够？今日次数用完？
    杀怪不给 XP 是否 AFK（2分钟没动）？区块累计是否超 500？10分钟是否超 1000？是否刷怪笼怪（0.5×）？
    出口价突然变低 该物品全服出口过多（饱和度）？或经济处于紧缩期？
    不能去下界/末地 下界需 T2（3000），末地需 T3（5000）。/tpa 也绕不过。
    挖矿没掉碎片 是否创造模式？2% 概率没触发？该方块不在补偿列表（石头/深板岩/凝灰岩）？
